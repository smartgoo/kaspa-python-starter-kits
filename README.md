# Kaspa Python Starter Kit

A collection of [Cookiecutter](https://cookiecutter.readthedocs.io/) templates for scaffolding Python projects that interact with the [Kaspa](https://kaspa.org/) blockchain.

## Available Templates

| Template | Description |
|----------|-------------|
| `kaspa_rest_api_starter` | FastAPI REST API with async Kaspa SDK integration, Pydantic validation, and OpenAPI docs |

## Quick Start

### Prerequisites

- Python 3.10+
- [Cookiecutter](https://cookiecutter.readthedocs.io/en/latest/installation.html)

```bash
pip install cookiecutter
```

### Generate a Project

```bash
# From GitHub
cookiecutter gh:smartgoo/python-kaspa-starter-kit

# From local directory
cookiecutter /path/to/python-kaspa-starter-kit
```

Follow the prompts to configure your project, or use `--no-input` for defaults.
